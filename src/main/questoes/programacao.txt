FÁCIL

1. Qual é o tipo de variável que pode mudar seu valor durante a execução do programa?
a) Variável constante
b) Variável local
c) Variável global
d) Variável tipada

Resposta: b) Variável local

2. Qual é o tipo de estrutura de controle que executa um bloco de código zero ou mais vezes?
a) Laço for
b) Laço while
c) Laço do-while
d) Laço switch

Resposta: b) Laço while

3. Qual é o método de ordenação de strings que compara caracteres por caractere?
a) Sort()
b) Reverse()
c) Toupper()
d) Tolower()

Resposta: a) Sort()

4. Qual é o tipo de erro que ocorre quando um programa tenta acessar um índice inválido de um array?
a) Erro de syntax
b) Erro de runtime
c) Erro de tipos
d) Erro de memória

Resposta: b) Erro de runtime

5. Qual é a linguagem de programação que é facilmente interpretada pelo browser?
a) Java
b) Python
c) JavaScript
d) C++

Resposta: c) JavaScript

6. Qual é o tipo de variável que tem um valor fixo e não pode ser alterado durante a execução do programa?
a) Variável local
b) Variável global
c) Variável constante
d) Variável tipada

Resposta: c) Variável constante

7. Qual é o tipo de estrutura de dados que é uma coleção ordenada de valores?
a) Vetor
b) Matriz
c) Pilha
d) Fila

Resposta: a) Vetor

8. Qual é o método de entrada/saída que permite ao usuário digitar uma string?
a) Console.ReadLine()
b) Console.Write()
c) Console.WriteLine()
d) Console.Read()

Resposta: a) Console.ReadLine()

9. Qual é o tipo de exceção que ocorre quando um programa tenta dividir por zero?
a) Exceção de divisão por zero
b) Exceção de tipo inválido
c) Exceção de falta de memória
d) Exceção de syntax

Resposta: a) Exceção de divisão por zero

10. Qual é a linguagem de programação que é utilizada para desenvolver aplicativos móveis?
a) Java
b) Python
c) C++
d) Swift

Resposta: d) Swift
